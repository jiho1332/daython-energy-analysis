# 데이톤 공모전 - 스마트팩토리 에너지 분석

AI 기술을 활용한 지속가능한 에너지·환경 솔루션 아이디어 공모전 (예선) 참가 프로젝트입니다.

## 공모 주제
제조업 생산 공장(스마트 팩토리)의 전력 및 설비 운영 데이터를 분석하여 에너지 사용 패턴, 피크 시간대, 비효율 구간을 도출하고, AI 기반 관리 모델 아이디어를 제안합니다.

## 데이터
- **원본 파일**: `rtu_data_full.csv` (약 4.57GB)
- **저장 위치**: 구글 드라이브 `데이톤 공모전` 폴더 (용량 문제로 GitHub에는 미포함, `.gitignore`로 제외)
- **전처리 파일**: `rtu_data_full.parquet` (동일 폴더에 저장, CSV 대비 로딩 속도 개선)

### 데이터 구조
- 5초 간격 측정, 2024-12-01 ~ 2025-04-30 (5개월치)
- 총 33,696,013행 × 19개 컬럼 (설비 정보, 전압/전류, 유효/무효전력, 역률, 누적전력량 등)
- 13개 설비(module): PM-3, 분전반, 4호기, 3호기, 2호기, 예비건조기, 호이스트, 6호기, L-1전등, 분쇄기 등
